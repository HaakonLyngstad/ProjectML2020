Boosting

This is an Ensemble Method. You can learn more about other ensemble methods bagging and stacking by clicking the links!

The idea behind boosting is that models should be able to learn form one another. If you have read about bagging (if not click the first link above), you know that each model in the bagging method learns on different data, but at the same time. They are trained in parallel.

Boosting works sequentially, that is, one after the other. Each iteration they learn from the prevoius model by identifying what that model did wrong, and adjusting its metrics to prioritize getting right the last models mistakes. This is done by doing something called re-weighting, which is adjusting the importance of data points according to what should be in focus, and what should not.
