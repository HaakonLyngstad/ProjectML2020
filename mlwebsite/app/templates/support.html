Help us improve!

Our machine learning model rely on large data sets to accuratly predict fradulent job postings. If you are using our platform to find fradulent job postings for a platform, please consider uploading any data you have available so we can improve this model.

Format

The data must be submitted as an csv file containg two rows. The first row should be titled "text", and contain a combination of the company description, the job description and the requirments for the job. The second column should be titled "fake", and contain information on wether the posting is fake or real. Real announcements is to be annotaded with a 0, and fake ones with a 1 in this column. For the form title, give the name of the company or page you represent. In the comments, give adequate contact info so that we can reach out for validation of your source. Thank you very much for conttibuting!
